# Personal Portfolio Website

## 📌 Project Overview
This project is a personal portfolio website developed as part of a Web Systems course. The website presents information about my academic background, personal interests, future career aspirations, and a reflection on my web design approach.

The goal of this project is to demonstrate foundational front-end development skills and apply UI design principles in creating a structured and responsive web page.

---

## 🎯 Features
- Personal introduction and background section  
- Career goals and future aspirations page  
- Reflection on web design and development process  
- Multi-section layout with navigation structure  
- Responsive design for improved user experience  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  

---

## 🧠 Key Learnings
Through this project, I developed and strengthened skills in:
- Structuring web pages using semantic HTML  
- Designing and styling responsive layouts using CSS  
- Organizing content for clarity and usability  
- Applying basic UI/UX design principles
